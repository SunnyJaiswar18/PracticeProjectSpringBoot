Java Objects and Classes

In Java, objects and classes are the core concepts of Object-Oriented Programming (OOP). A class is a blueprint for creating objects, while an object represents an instance of a class. An object has a state, behavior, and identity.

What is an Object in Java?

An object is an entity that has state and behavior. It represents real-world entities like a chair, car, or bank account. For example, a pen has a state (color, brand) and behavior (write). An object is an instance of a class, and each object has its unique identity.

    class Pen {
        String brand = "Reynolds";
        String color = "White";
        
        void write() {
            System.out.println("Writing...");
        }
    }

What is a Class in Java?

A class is a group of objects with common properties. It is a template or blueprint from which objects are created. A class can contain fields, methods, constructors, and more.

    class Student {
        int id;
        String name;
        
        void display() {
            System.out.println(id + " " + name);
        }
    }

Example: Defining a Class and Object

In this example, we create a class Student with fields id and name, then instantiate an object of the Student class to print its values.

    class Student {
        int id;
        String name;
    }

    public class TestStudent {
        public static void main(String[] args) {
            Student s1 = new Student();
            System.out.println(s1.id);  // Outputs: 0
            System.out.println(s1.name);  // Outputs: null
        }
    }

Example: Initializing an Object through a Reference Variable

We can initialize an object by assigning values through the reference variable as shown in the following example:

    class Student {
        int id;
        String name;
    }

    public class TestStudent {
        public static void main(String[] args) {
            Student s1 = new Student();
            s1.id = 101;
            s1.name = "Sonoo";
            System.out.println(s1.id + " " + s1.name);  // Outputs: 101 Sonoo
        }
    }

Example: Initialization through Method

Objects can also be initialized using methods. Below, the insertRecord method initializes the objects, and the displayInformation method displays the values.

    class Student {
        int rollno;
        String name;
        
        void insertRecord(int r, String n) {
            rollno = r;
            name = n;
        }

        void displayInformation() {
            System.out.println(rollno + " " + name);
        }
    }

    public class TestStudent {
        public static void main(String[] args) {
            Student s1 = new Student();
            Student s2 = new Student();
            s1.insertRecord(111, "Karan");
            s2.insertRecord(222, "Aryan");
            s1.displayInformation();  // Outputs: 111 Karan
            s2.displayInformation();  // Outputs: 222 Aryan
        }
    }

Example: Initialization through Constructor

We can also initialize objects using constructors, which are special methods used to set the initial state of an object.

    class Student {
        int rollno;
        String name;

        Student(int r, String n) {
            rollno = r;
            name = n;
        }

        void displayInformation() {
            System.out.println(rollno + " " + name);
        }
    }

    public class TestStudent {
        public static void main(String[] args) {
            Student s1 = new Student(111, "Karan");
            Student s2 = new Student(222, "Aryan");
            s1.displayInformation();  // Outputs: 111 Karan
            s2.displayInformation();  // Outputs: 222 Aryan
        }
    }

Example: Anonymous Object

An anonymous object is an object without a reference variable. It can be used only once, typically when you don't need to access the object again after its use.

    class Calculation {
        void fact(int n) {
            int fact = 1;
            for (int i = 1; i <= n; i++) {
                fact = fact * i;
            }
            System.out.println("Factorial is " + fact);
        }

        public static void main(String[] args) {
            new Calculation().fact(5);  // Anonymous object to calculate factorial
        }
    }

Example: Real World Example - Account Class

In the following example, we demonstrate a banking system where we deposit and withdraw money from an account. This is a real-world example of using objects to model a system.

    class Account {
        int acc_no;
        String name;
        float amount;

        void insert(int a, String n, float amt) {
            acc_no = a;
            name = n;
            amount = amt;
        }

        void deposit(float amt) {
            amount = amount + amt;
            System.out.println(amt + " deposited");
        }

        void withdraw(float amt) {
            if (amount < amt) {
                System.out.println("Insufficient Balance");
            } else {
                amount = amount - amt;
                System.out.println(amt + " withdrawn");
            }
        }

        void checkBalance() {
            System.out.println("Balance is: " + amount);
        }

        void display() {
            System.out.println(acc_no + " " + name + " " + amount);
        }
    }

    public class TestAccount {
        public static void main(String[] args) {
            Account a1 = new Account();
            a1.insert(832345, "Ankit", 1000);
            a1.display();  // Outputs: 832345 Ankit 1000.0
            a1.checkBalance();  // Outputs: Balance is: 1000.0
            a1.deposit(40000);
            a1.checkBalance();  // Outputs: Balance is: 41000.0
            a1.withdraw(15000);
            a1.checkBalance();  // Outputs: Balance is: 26000.0
        }
    }
