Methods in Java

A method in Java is a block of code that performs a specific task or operation. Methods help in achieving code reusability and easy code modification.

    // A simple method declaration
    public static void findEvenOdd(int num) {
        if (num % 2 == 0)
            System.out.println(num + " is even");
        else
            System.out.println(num + " is odd");
    }

Example: Calling a User-defined Method

Here is an example of calling the method findEvenOdd that checks if a number is even or odd:

    public static void main(String[] args) {
        Scanner scan = new Scanner(System.in);
        System.out.print("Enter the number: ");
        int num = scan.nextInt();
        findEvenOdd(num); // Method call
    }

Example: Returning a Value from a Method

This example shows how a method can return a value. Here, the method add returns the sum of two integers:

    public static int add(int n1, int n2) {
        int sum = n1 + n2;
        return sum;
    }

    public static void main(String[] args) {
        int a = 5, b = 10;
        int result = add(a, b); // Method call
        System.out.println("The sum is: " + result);
    }

Example: Static Methods

A static method belongs to the class, not an instance of the class. It can be called without creating an object of the class:

    static void show() {
        System.out.println("This is a static method.");
    }

    public static void main(String[] args) {
        show(); // Static method call
    }

Example: Instance Methods

An instance method requires an object of the class to be called:

    public int add(int a, int b) {
        return a + b;
    }

    public static void main(String[] args) {
        InstanceMethodExample obj = new InstanceMethodExample();
        System.out.println("The sum is: " + obj.add(12, 13));
    }

Accessor and Mutator Methods

Accessor methods get the value of a private field, while mutator methods modify the value:

    public class Student {
        private int roll;
        private String name;

        // Accessor method
        public int getRoll() {
            return roll;
        }

        // Mutator method
        public void setRoll(int roll) {
            this.roll = roll;
        }
    }

Example: Abstract Method

Abstract methods do not have a body. They are declared in abstract classes and must be implemented by subclasses:

    abstract class Demo {
        abstract void display();
    }

    public class MyClass extends Demo {
        void display() {
            System.out.println("Abstract method implemented.");
        }

        public static void main(String args[]) {
            Demo obj = new MyClass();
            obj.display();
        }
    }

Example: Factory Method

A factory method creates and returns an instance of a class:

    public class NumberFormatExample {
        public static NumberFormat getNumberInstance() {
            return NumberFormat.getInstance();
        }

        public static void main(String[] args) {
            NumberFormat nf = getNumberInstance();
            System.out.println(nf.format(12345.6789));
        }
    }
