Java This Keyword

The this keyword in Java is a reference variable that refers to the current object. It can be used in various ways to refer to the current object and resolve ambiguities.

Example 1: Using this to Refer to Instance Variables

When local variables (parameters) have the same names as instance variables, this can be used to distinguish between them:

class Student{  
    int rollno;  
    String name;  
    float fee;  
    Student(int rollno, String name, float fee){  
        this.rollno = rollno;  
        this.name = name;  
        this.fee = fee;  
    }  
    void display(){  
        System.out.println(rollno + " " + name + " " + fee);  
    }  
}

class TestThis1{  
    public static void main(String args[]){  
        Student s1 = new Student(111, "Ankit", 5000f);  
        Student s2 = new Student(112, "Sumit", 6000f);  
        s1.display();  
        s2.display();  
    }  
}

Output:

111 Ankit 5000.0

112 Sumit 6000.0

Example 2: Using this to Invoke Current Class Methods

The this keyword can be used to invoke the current class method:

class A{  
    void m(){  
        System.out.println("hello m");  
    }  
    void n(){  
        System.out.println("hello n");  
        this.m();  // invokes current class method m()
    }  
}  
class TestThis4{  
    public static void main(String args[]){  
        A a = new A();  
        a.n();  
    }  
}

Output:

hello n

hello m

Example 3: Using this() to Invoke Current Class Constructor

The this() constructor call can be used to reuse constructors in a class (constructor chaining):

class A{  
    A(){  
        System.out.println("hello a");  
    }  
    A(int x){  
        this();  
        System.out.println(x);  
    }  
}  
class TestThis5{  
    public static void main(String args[]){  
        A a = new A(10);  
    }  
}

Output:

hello a

10

Example 4: Using this as an Argument in a Method

The this keyword can be passed as an argument in the method. This is particularly useful in event handling:

class S2{  
    void m(S2 obj){  
        System.out.println("method is invoked");  
    }  
    void p(){  
        m(this);  // passing current object as argument
    }  
    public static void main(String args[]){  
        S2 s1 = new S2();  
        s1.p();  
    }  
}

Output:

method is invoked

Example 5: Using this as an Argument in the Constructor Call

You can also pass this as an argument in the constructor. This is helpful when objects of one class are passed to another class:

class B{  
    A4 obj;  
    B(A4 obj){  
        this.obj = obj;  
    }  
    void display(){  
        System.out.println(obj.data);  
    }  
}  

class A4{  
    int data = 10;  
    A4(){  
        B b = new B(this);  
        b.display();  
    }  
    public static void main(String args[]){  
        A4 a = new A4();  
    }  
}

Output:

10

Example 6: Using this to Return the Current Class Instance

In some cases, you can use this to return the current instance of the class from a method:

class A{  
    A getA(){  
        return this;  // returning current class instance
    }  
    void msg(){  
        System.out.println("Hello java");  
    }  
}  

class Test1{  
    public static void main(String args[]){  
        new A().getA().msg();  
    }  
}

Output:

Hello java

Example 7: Proving this Refers to Current Class Instance

To prove that this refers to the current class instance, here is an example where the same reference ID is printed for both the object and this:

class A5{  
    void m(){  
        System.out.println(this);  // prints the reference ID
    }  
    public static void main(String args[]){  
        A5 obj = new A5();  
        System.out.println(obj);  // prints the reference ID  
        obj.m();  
    }  
}

Output:

A5@22b3ea59

A5@22b3ea59
